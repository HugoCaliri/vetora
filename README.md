# 📐 Vetora - Visualizador Interativo de Álgebra Linear

Uma aplicação web desenvolvida para auxiliar estudantes na aprendizagem de Álgebra Linear por meio de visualizações interativas e representações gráficas em tempo real.

## 📖 Sobre o Projeto

O **Visualizador Interativo de Álgebra Linear** foi desenvolvido como uma ferramenta didática para apoiar o ensino dos principais conceitos da disciplina de Álgebra Linear da Universidade Federal do Amazonas (UFAM).

A aplicação permite que o usuário manipule vetores, matrizes e parâmetros matemáticos de forma interativa, observando instantaneamente seus efeitos geométricos no plano cartesiano.

O objetivo é complementar o ensino tradicional, tornando conceitos abstratos mais intuitivos através da visualização gráfica.

---

## ✨ Funcionalidades

A aplicação é dividida em cinco módulos:

### 🔹 Módulo 1 — Vetores Básicos

- Soma de vetores
- Norma (módulo)
- Produto interno
- Ângulo entre vetores
- Distância Euclidiana
- Projeção ortogonal
- Visualização gráfica em tempo real

---

### 🔹 Módulo 2 — Combinação Linear

- Combinação linear de vetores
- Ajuste dos coeficientes α e β
- Span dos vetores
- Verificação de dependência e independência linear
- Visualização do método do paralelogramo

---

### 🔹 Módulo 3 — Transformações Lineares

- Matrizes 2×2
- Aplicação da transformação em vetores
- Transformações pré-definidas:
  - Identidade
  - Rotação
  - Reflexão
  - Homotetia
  - Cisalhamento
- Cálculo do determinante
- Verificação de invertibilidade

---

### 🔹 Módulo 4 — Operadores Lineares

- Cálculo de autovalores
- Cálculo de autovetores
- Visualização de autoespaços
- Subespaços invariantes
- Representação gráfica da ação do operador

---

### 🔹 Módulo 5 — Produto Interno e Gram-Schmidt

- Ortogonalidade
- Produto interno
- Processo de Gram-Schmidt
- Construção de base ortonormal
- Visualização dos vetores ortogonais

## ▶️ Como Executar

### 1. Clone o repositório

```bash
git clone https://github.com/HugoCaliri/vetora.git
```

### 2. Entre na pasta

```bash
cd vetora
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Execute o projeto

```bash
npm start
```
